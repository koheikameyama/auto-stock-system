## 背景

マルチアセット分散の文脈で、金ETFを現行 auto-stock-trader の戦略基盤に組み込めるかを検証。
レジーム別検証（2026-04-17）で A期(平穏ボックス) と E期(直近急落) のDD緩和余地が示唆されており、

## 結論: 金ETF は現行戦略の universe に追加しない

### 構造的に発火しない理由

1. **GU の gap≥3% シグナル**: 金ETF は10年で gap≥3% の寄付き日がほぼゼロ。日次値動きのstddev 1.05% に対し3%は3σ超えのテールイベント
2. **PSC の +15%急騰**: 金ETF は20日で15%動くことが極めて稀。10年でも該当期間なし
3. **値動き性質の根本的な違い**: 金 = マクロ要因（金利・ドル・地政学）主導 / 中小型株 = ミクロ要因（出来高サージ・モメンタム）主導 の検証時の仮説が完全に裏付けられた

### この検証で確定したこと

- **「現行GU/PSC戦略は中小型株固有のエッジ」が定量確認された**: 同戦略を別アセットに転用してもエッジを抽出できないと予想すべき
- **マルチアセット分散による株式DD緩和は、金ETF + 現行戦略 の組み合わせでは実現不可能**: A期/E期のDD緩和には別アプローチが必要
- **金で本格運用するには金専用戦略（トレンドフォロー / Donchian breakout / RSI mean-reversion 等）の新規設計が必要**: ROIは低い（金単体のSharpeは0.3〜0.5が一般的）

## 今後のアンチパターン（再検証禁止）

- 金ETF（1326.T / 1540.T）を現行 universe に追加してGU/PSC を回す → **シグナル0件で時間の無駄**
- 同じ理由で他の低ボラETF（債券ETF・ディフェンシブETF）も既存戦略と相性が悪い可能性が高い

## プロの教訓

**「同じ戦略基盤を別アセットに転用すれば自動でマルチアセット分散になる」は誤り。**
- 戦略のエッジはアセット固有の値動き性質（ボラ・モメンタム継続性・ニュース反応速度）に依存
- マルチアセット分散したい場合は、各アセットに合った戦略を別個に設計するのが正攻法
- 既存基盤の流用 ≠ コスト削減（むしろ「使えないことを確認するBT」のコストが発生）

## 金ETF戦略が経済合理性を持つ資金規模（参考資料）

「金専用戦略を新設するなら、いくらから機能するか」を試算。
本検証で「現行戦略では金が機能しない」が確定した上で、**金専用戦略を作るとしても本当に必要か**を判断するための資料。

### 想定する金専用戦略のスペック

| 指標 | 値（一般的な金トレンドフォロー戦略） |
|---|---:|
| Sharpe比 | 0.3〜0.5 |
| 期待年率リターン | +5〜10% |
| 期待最大DD | 15〜25% |
| 期待 Calmar比 | **0.3〜0.7**（中小型株GU/PSC の Calmar 9.37 と比較すると桁違いに弱い） |
| 想定トレード頻度 | 年4〜10件（保有1〜3ヶ月） |

### コスト構造（1326.T、立花証券スタンダードプラン想定）

| コスト項目 | ¥1Mトレード時 | 比率 |
|---|---:|---:|
| 手数料 | ¥495 | 0.05% |
| スプレッド | ~¥1,000 | 0.10% |
| 信託報酬（年率） | 0.44% | - |
| 譲渡益税（勝ち時のみ） | 利益×20% | - |

**保有3ヶ月のトータル実効コスト ≈ 0.4〜0.5%/trade**

### 資金規模別の経済性（リターン+7%×Calmar 0.5 を仮定）

| 資金 | 1トレード金額 | 想定年間損益 | 判定 |
|---:|---:|---:|---|
| ¥1M | ¥200K | +¥14K | **不可**（コスト3%/年で完全に消える） |
| ¥3M | ¥600K | +¥42K | 不可 |
| **¥10M** | ¥2M | **+¥140K** | 損益分岐点 |
| **¥30M** | ¥6M | +¥420K | **最低ライン** |
| **¥50M** | ¥10M | +¥700K | **実用域** |
| ¥100M+ | ¥20M+ | +¥1.4M+ | 流動性制約が出始める |

### なぜ¥10M未満で機能しないか

1. **固定コスト（手数料・スプレッド）が相対的に重い**: 1326.T はスプレッド数千円で動くため、¥1Mトレードでは0.1%、¥100Kトレードでは1%の負担に膨れる
2. **ポジションサイズが整わない**: リスク2%/trade で SL ATR×2.0（金で約4%）を考えると、¥1M資金では1326.T を **1株しか買えない**
3. **複利効果が無視できる**: +¥14K/年 × 3年 = +¥42K では検証期間中に運用判断できない（ノイズと区別不可）

### 同じ¥30M〜¥50M に達したときの代替選択肢との ROI 比較

| 選択肢 | 期待Calmar寄与 | 実装難度 |
|---|---:|---:|
| 金ETFトレンドフォロー新設 | +0.1〜0.3（全体DD緩和） | 高 |
| **米国株 (auto-us-stock-trader) 本番化** | **+5〜8** | 中 |
| **FX (auto-fx-trader) 戦略確立** | +2〜4 | 中 |
| 暗号通貨 (auto-crypto-trader) 戦略確立 | +1〜3 | 中 |

→ **¥30M〜¥50M レベルでも、金より米国株/FX に投資する方がROI 5〜20倍高い**。

### 運用規模別の推奨アクション

| 運用規模 | 推奨 |
|---|---|
| **〜¥5M（現状）** | 金は完全に放置。米国株/FX/crypto を優先 |
| ¥5M〜¥30M | 金は引き続き放置。米国株マルチアセット化を進める |
| ¥30M〜¥100M | 金専用戦略の設計を検討開始（PoCレベル） |
| ¥100M+ | 金 + 米国株 + FX のマルチアセット運用が経済合理性を持つ |

### 結論（資金規模目線）

**「金ETF戦略は¥30M以上の運用規模で初めて検討に値する。それ未満では中小型株エッジ（Calmar 9.37）に集中投下する方が圧倒的にROIが高い」**

現在の auto-stock-trader 運用想定（¥500K〜¥5M）では、**金は少なくとも今後5〜10年は不要**。

### 試算の前提（再評価が必要になる条件）

以下のいずれかが変わったら本資料の数値を再試算する必要がある:

1. **金単体のSharpeが大幅改善**（マクロ環境の構造変化、AI戦略でエッジ発見、等）
2. **取引コストの劇的低下**（手数料無料化、スプレッド縮小、等）
3. **株式戦略のCalmarが2-3に低下**（マルチアセット分散の必要性が高まる）
4. **保有資産が¥50Mを超える**（絶対リターン額が運用判断に意味を持つ規模）
