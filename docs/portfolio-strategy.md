# ハイブリッド運用戦略（日本株システム + 米国株）

## 推奨ポートフォリオ

### **フェーズ1（1-2年目）：システム実証期間**

```
日本株システム：50万円（100%）
米国株：0円

目的：
- システムの安定性確認
- 年利103%を実証
- データ蓄積
```

**結果：1年後に約100万円**

---

### **フェーズ2（3年目〜）：ハイブリッド運用開始**

```
総資産：100万円（2年間の利益含む）

配分：
- 日本株システム：30万円（30%）
- 米国株インデックス：30万円（30%）
- 現金（待機資金）：40万円（40%）

運用：
- 日本株：年利103%（実働10%想定）→ 年+30万
- 米国株：年利8%（長期）→ 年+2.4万
- 現金：必要に応じて追加投資

合計年利：32.4% on 100万円
```

---

### **フェーズ3（5年目〜）：段階的拡大**

```
総資産：200万円（累積利益含む）

配分：
- 日本株システム：50万円（上限維持）
- 米国株インデックス：100万円
- 現金/債券：50万円

運用：
- 日本株：年+50万（年利103%）
- 米国株：年+8万（年利8%）
- 合計：年+58万（総資産年利29%）
```

## リバランスルール

### **四半期チェック（3ヶ月ごと）**

1. **日本株システム**
   - 30万円未満：米国株から補填
   - 30-50万円：維持
   - 50万円超：超過分を米国株へ移動

2. **米国株**
   - 基本的に長期保有
   - 大幅な上昇時（+30%超）は一部利確検討

3. **現金**
   - 緊急バッファー：常時5-10万円
   - それ以上は投資に回す

---

## 税金の考慮

### **日本株システム**
- 譲渡益税：20.315%
- 年間利益50万円 → 税金10万円

### **米国株**
- 外国税額控除を活用
- 配当：米国10% + 日本20.315%（二重課税調整）
- 譲渡益：日本20.315%

---

## 結論

**3年目以降、資金が100万円を超えたらハイブリッド運用を推奨**

- 日本株システム：30-50万円（高年利103%を維持）
- 米国株：30-100万円（安定成長7-10%）
- 現金：緊急バッファー

**これにより：**
- 資金効率向上
- リスク分散
- 長期資産形成

**最初の2年間は日本株システムのみで実証してから移行することを推奨。**

## 関連分析スクリプト

この運用戦略の裏付けとなる数学的分析を行うスクリプトが `scripts/` に用意されています。

### **1. analyze-peak-positions.ts** - 同時保有数分析

**目的**: バックテスト結果から最大同時保有銘柄数を分析し、必要資金を見積もる

**使い方**:
```bash
npx tsx scripts/analyze-peak-positions.ts
```

**出力内容**:
- ピーク同時保有数（例：5枠）
- 平均同時保有数（例：1.8枠）
- 同時保有数の分布ヒストグラム
- ピーク時の資金配分詳細

**活用例**:
- 「初期資金50万円で足りるか？」の検証
- リスク管理（分散しすぎていないか確認）

---

### **2. compare-compound-vs-fixed.ts** - 複利 vs 固定資金の比較

**目的**: 複利運用（利益再投資）と固定資金運用（50万固定で利益出金）を10年間シミュレーション

**背景**:
- バックテストで「50万円=年利103%、75万円=年利30%」が判明
- 資金が増えると年利が激減する特性を持つ
- 「複利は常に有利」という常識が当てはまらない可能性

**使い方**:
```bash
npx tsx scripts/compare-compound-vs-fixed.ts
```

**出力内容**:
- 年次推移表（固定戦略 vs 複利戦略）
- 出金累計額
- 10年後の資産総額比較

**重要な発見**:
このシステムでは「固定50万円で高年利を維持し、利益を出金する戦略」が数学的に優位。

---

### **3. find-breakeven-point.ts** - 損益分岐点の精密計算

**目的**: 「何年目で複利が固定資金を逆転するか」を実測データ（7パターン）で計算

**実測データ**:
- 50万円 → 年利103.1%
- 75万円 → 年利90.3%
- 100万円 → 年利81.7%
- 150万円 → 年利80.3%
- （500万円まで計測）

**使い方**:
```bash
npx tsx scripts/find-breakeven-point.ts
```

**出力内容**:
- 15年間の年次推移
- 損益分岐点の年数と資金規模
- 資金規模別の年利・年間利益比較表

**戦略への示唆**:
- 短期（1-7年）：固定50万円が有利
- 中期（8-10年）：複利が逆転し始める
- 長期（11年以降）：複利が圧倒的に有利
- **推奨**: 3-5年は固定戦略で実績を作り、その後判断

---

## これらの分析が示す結論

1. **初期資金**: 50万円あれば十分（ピーク5枠保有でも対応可能）
2. **運用方針**: 最初の3-5年は固定50万円で年利103%を維持
3. **利益の使い道**: 出金して米国株インデックスへ分散投資
4. **長期戦略**: 日本株は50万円上限、それ以上は米国株で運用

この数学的根拠に基づき、上記のハイブリッド運用戦略を提案しています。
