# 銘柄比較 仕様書

## 概要

2銘柄のデータを並べて比較する機能です。スマートスイッチ（乗り換え提案）の代替として、ユーザー自身が銘柄を比較して判断できる環境を提供します。

> **注意**: データの比較表示のみ行い、どちらが良いかの推奨は含みません。投資判断はユーザー自身が行います。

**ページパス**: `/stocks/compare?ids=xxx,yyy`

## 画面構成

### 銘柄選択

- 2つの銘柄選択フィールド
- 銘柄検索（証券コード / 銘柄名）
- ポートフォリオ / ウォッチリストからの選択も可能

### 比較テーブル

以下のカテゴリでデータを並べて表示:

#### 基本情報

| 項目 | 説明 |
|------|------|
| 銘柄名・コード | 基本情報 |
| セクター | 所属セクター |
| 時価総額 | 時価総額 |
| 現在価格 | リアルタイム価格 |

#### 株価パフォーマンス

| 項目 | 説明 |
|------|------|
| 日次変化率 | 前日比(%) |
| 週間変化率 | 週間の変化率(%) |
| 52週高値/安値 | 年間の高値/安値 |
| ボラティリティ | 価格変動の大きさ |

#### テクニカル指標

| 項目 | 説明 |
|------|------|
| RSI (14) | 相対力指数 |
| MACD | 移動平均収束拡散 |
| MA乖離率 | 25日移動平均線からの乖離 |
| 出来高比率 | 20日平均出来高比 |

#### 財務指標

| 項目 | 説明 |
|------|------|
| PER | 株価収益率 |
| PBR | 株価純資産倍率 |
| ROE | 自己資本利益率 |
| 配当利回り | 年間配当/株価 |
| EPS | 1株当たり利益 |
| 負債比率 | D/E Ratio |
| 売上成長率 | 前年比 |
| 純利益成長率 | 前年比 |

#### 分析スコア（レポートがある場合）

| 項目 | 説明 |
|------|------|
| テクニカルスコア | 0-100 |
| ファンダメンタルスコア | 0-100 |
| 財務健全性ランク | A-E |
| 市場シグナル | bullish / neutral / bearish |

### 比較チャート

- 2銘柄の株価推移を1つのチャート上に重ねて表示
- 期間初日を0%としたパフォーマンス比較
- 期間選択: 1ヶ月 / 3ヶ月 / 1年
