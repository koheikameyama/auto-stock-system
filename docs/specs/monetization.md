# 収益化（チャット課金プラン） 仕様書

## 概要

AIチャット機能に「深さ × 回数」の制限を設け、Free / Premium の2プランで収益化する。

- **Free**: 簡潔な回答（ツールなし）、1日5回まで
- **Premium**: 詳細分析（ツール連携 + データ付き）、1日50回まで

> **設計方針**: チャット以外の機能（ダッシュボード、スクリーニング、ポートフォリオ管理等）はすべて無料のまま。課金対象はAIチャットの「深さ」と「回数」のみ。

## プラン比較

| 項目 | Free | Premium（月額980円） |
|------|------|---------------------|
| **チャット回数** | 5回/日 | 50回/日 |
| **回答の深さ** | 簡潔回答（ツールなし） | 詳細分析（ツール連携 + データ付き） |
| **モデル** | GPT-4o-mini | GPT-4o-mini |
| **チャット以外の機能** | すべて利用可能 | すべて利用可能 |

### 回答の深さの違い

#### Free（ツールなし）

- システムプロンプト + ユーザーの質問のみで回答
- 銘柄コンテキスト（プリロードデータ）は使用する
- ツール呼び出し（`getStockPrice`, `getPortfolioSummary` 等）は行わない
- 一般的な知識ベースの回答

**回答例（Free）**:
```
トヨタ自動車は日本を代表する自動車メーカーです。
PER（株価収益率）は約10倍で、自動車セクターの平均と比較すると割安圏にあります。

💡 より詳しいリアルタイムデータや分析をご希望の場合は、
プレミアムプランをご検討ください。
```

#### Premium（ツール連携あり）

- ツール呼び出しでリアルタイムデータを取得して回答
- 財務指標、AI分析、ニュース、ポートフォリオ分析を統合
- 具体的な数値に基づいた詳細な分析

**回答例（Premium）**:
```
トヨタ自動車（7203）の現在の状況をまとめます。

📊 現在値: 2,850円（前日比 +1.2%）
📈 テクニカル: RSI 62 — やや強気圏（70以上で過熱）
💰 PER: 9.8倍 / PBR: 1.1倍 — セクター平均より割安
📰 直近ニュース: EV戦略の加速が報じられ、市場は好感

短期トレンドは上昇基調ですが、為替（ドル円）の変動が
業績に影響するため注意が必要です。
```

## 回数制限

### カウント方法

- **ユーザーの送信1回 = 1カウント**（AIの応答はカウントしない）
- カウントはJST 00:00にリセット
- 制限に達した場合、チャット入力を無効化し、アップグレード案内を表示

### 制限値

```typescript
// lib/constants.ts
export const CHAT_PLAN = {
  FREE_DAILY_LIMIT: 5,      // Freeプラン: 1日5回
  PREMIUM_DAILY_LIMIT: 50,  // Premiumプラン: 1日50回
} as const;
```

## データモデル

### ChatUsage テーブル（新規）

ユーザーごとの日次チャット利用回数を記録する。

```prisma
model ChatUsage {
  id        String   @id @default(cuid())
  userId    String
  date      DateTime @db.Date
  count     Int      @default(0)
  user      User     @relation(fields: [userId], references: [id], onDelete: Cascade)
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt

  @@unique([userId, date])
  @@index([userId])
}
```

### User テーブル（既存フィールドを活用）

```prisma
// 既にスキーマに存在するフィールド
subscriptionPlan   String    @default("free") // 'free' | 'premium'
subscriptionExpiry DateTime?
```

## API仕様

### チャット API の変更（`POST /api/chat`）

既存のチャットAPIに以下のチェックを追加:

1. ユーザーの `subscriptionPlan` を取得
2. `ChatUsage` から今日の利用回数を取得
3. 制限超過の場合は 429 を返す
4. プランに応じてツールの有無を切り替え
5. 正常時は利用回数をインクリメント

**レスポンス（制限超過時）**:
```json
{
  "error": "CHAT_LIMIT_EXCEEDED",
  "message": "本日のチャット利用回数の上限に達しました",
  "limit": 5,
  "plan": "free"
}
```

### チャット残回数 API（新規）

`GET /api/chat/usage`

チャットUIで残り回数を表示するためのAPI。

**レスポンス**:
```json
{
  "plan": "free",
  "dailyLimit": 5,
  "used": 3,
  "remaining": 2
}
```

## UI変更

### チャットウィンドウ

- 残り回数をチャット入力欄の上に表示: 「残り3回 / 5回」
- 制限到達時: 入力欄を無効化し、「プレミアムにアップグレード」ボタンを表示
- Premiumユーザー: 残り回数を表示するが、50回なので実質気にならない

### アップグレード案内

制限到達時のみ、以下を表示:

```
📊 本日のチャット利用回数の上限に達しました

プレミアムプラン（月額980円）にアップグレードすると:
✅ 1日50回までチャットできます
✅ リアルタイムデータを使った詳細分析が受けられます

[プレミアムにアップグレード]
```

## コスト試算

### Free ユーザー（ツールなし）

- 入力: ~1,000トークン（システムプロンプト + 質問）
- 出力: ~500トークン（簡潔回答）
- **1回あたり約0.02円**
- 5回/日 × 0.02円 = **0.1円/日/ユーザー**

### Premium ユーザー（ツール連携あり）

- 入力: ~3,000〜5,000トークン（プリロード + ツール結果）
- 出力: ~2,000トークン（詳細回答）
- **1回あたり約0.3〜0.5円**
- 50回/日 × 0.5円 = **最大25円/日/ユーザー**
- 月額: 最大750円 → **980円で黒字**
